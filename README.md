# Data Transformer Project

## Tables Used
1. Customers
   - CustomerID
   - FirstName
   - LastName
   - Email
   - RegistrationDate

2. Orders
   - OrderID
   - CustomerID
   - OrderDate
   - TotalAmount

3. Employees
   - EmployeeID
   - FirstName
   - LastName
   - Department
   - HireDate
   - Salary

## Tasks Performed
- Created database and tables
- Inserted sample data
- Performed INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL OUTER JOIN
- Used subqueries for average calculations
- Extracted year and month from date
- Calculated date difference
- Formatted date
- Used CONCAT, REPLACE, UPPER, LOWER, and TRIM functions
- Calculated running total using window function
- Ranked orders using RANK()
- Applied CASE statement for discount calculation
- Categorized employee salaries as High, Medium, or Low

## How to Run
1. Open MySQL Workbench or any SQL tool.
2. Copy and paste the SQL script file.
3. Execute the script.
4. Run the SELECT queries to view results.
